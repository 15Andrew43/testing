1. При нажатии пользователем на кнопку  "История заказов" происходит:
	- изменятеся url на /orders/1
	- Сразу после этого отображается getorderhistory.html, который получает данные с помощью, например, метода getProducts() класса Order. Данные по продуктам хранятся в бд (с помощью таких штук как, например, @Column).
2. При нажатии пользователем на кнопку "создать заказ" происходит:
	- изменятеся url на /create/1
	- Сразу после этого отображается сreateorderpage.html. При нажатии на "Добавить продукт" происходит:
		- вызывается BrowserClient.prototype.addProduct, определенный в том же файле сreateorderpage.html.
	При нажатии на "Отправить" происходит:
		- вызывается BrowserClient.prototype.createOrder, отпределеннйы в том же файле сreateorderpage.html. createOrder делает проверки на корректность, а потом делает ajax-запрос.
3. При нажатии на "найти" (после предварительного ввода номера заказа) происходит:
	- изменятеся url на /order/1?clientId=1
	- Сразу после этого отображается сreateorderpage.html, отображающий информацию по Order, которая берется из бд.
4. При нажатии на "получить заказ" происходит:
	- ajax запрос, изменяющий бд
	- обновляется страничка с обновленной информацией
